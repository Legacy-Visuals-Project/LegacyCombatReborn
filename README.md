## LegacyCombatReborn

A spigot plugin to restore 1.8/1.7 era combat in 1.21.5+.

## Notes

This plugin does not support ``/reload``, if you use it expect things to cause issues or not function properly!

## Build Instructions

``./gradlew shadowJar``

Outputted jar is in ``build/libs/``
